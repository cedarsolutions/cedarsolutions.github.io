Official Cedar Backup Extensions

This package provides official Cedar Backup extensions. These are Cedar Backup actions that are not part of the "standard" set of Cedar Backup actions, but are officially supported along with Cedar Backup.

Author: Kenneth J. Pronovici <pronovic@ieee.org>

Submodules [hide private]
  • CedarBackup2.extend.amazons3: Store-type extension that writes data to Amazon S3.
  • CedarBackup2.extend.capacity: Provides an extension to check remaining media capacity.
  • CedarBackup2.extend.encrypt: Provides an extension to encrypt staging directories.
  • CedarBackup2.extend.mbox: Provides an extension to back up mbox email files.
  • CedarBackup2.extend.mysql: Provides an extension to back up MySQL databases.
  • CedarBackup2.extend.postgresql: Provides an extension to back up PostgreSQL databases.
  • CedarBackup2.extend.split: Provides an extension to split up large files in staging directories.
  • CedarBackup2.extend.subversion: Provides an extension to back up Subversion repositories.
  • CedarBackup2.extend.sysinfo: Provides an extension to save off important system recovery information.
